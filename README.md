Saraka
Saraka est une plateforme web de partage, de don, de location et de brocante d’objets. Saraka est développée avec le framework PHP Symfony.
Le projet s’inscrit dans une démarche écologique, économique et solidaire visant à encourager la réutilisation des objets, réduire le gaspillage , les déchets et resserrer la cohésion sociale entre particuliers.
Le nom Saraka fait référence à l’idée de partage généreux et de transmission présente dans plusieurs cultures africaines. La plateforme a été pensée comme une alternative plus responsable à la surconsommation, en offrant une seconde vie aux objets du quotidien.
Elle encourage une manière plus durable de consommer, où les objets peuvent continuer à être utiles plutôt que d’être rapidement remplacés.
Fonctionnalités principales
    • Authentification sécurisée des utilisateurs 
    • Gestion complète des annonces (CRUD : Create, Read, Update , Delete) 
    • Publication d’articles avec plusieurs images 
    • Système de transaction (achat, location, don, brocante) 
    • Recherche et navigation par type d’annonce 
    • Messagerie entre utilisateurs 
    • Gestion des profils utilisateurs 
    • Interface responsive adaptée aux différents écrans

Architecture du projet
Le projet repose sur une architecture MVC (Modèle Vue Controller) permettant une séparation claire entre :
    • la logique métier, 
    • la gestion des données, 
    • et l’interface utilisateur. 
Les principales entités du projet sont :
    • Users 
    • Articles 
    • Images 
    • Catégories 
    • Conversations 
    • Messages 
Objectifs du projet
Ce projet m’a permis de passer d’une approche principalement frontend à une vision full stack du développement web.
Il m’a également permis d’approfondir :
    • la logique backend avec Symfony, 
    • la conception et gestion des tables et  bases de données relationnelles, 
    • la conception d’interfaces responsives, avec Boostrap, JS et CSS personnalisé,
    • ainsi que l’organisation d’un projet web structuré et évolutif.
